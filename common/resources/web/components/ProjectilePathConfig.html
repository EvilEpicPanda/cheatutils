Projectile Path

Show most likely path of projectiles (Minecraft adds some random to every shot).
Displayed path in not precise in close distance, since player shoots projectile from eyes, but module draws path from right hand.
This is done to have better visuals, in other case you would just see single vertical line.
Show path for:
Traces Duration:
In seconds, how long trace remains visible.
Fade Duration:
In seconds. Last X seconds when trace is visible it starts to gradually fade out.
